Neuromancer Neurostim v1.0 Help

Introduction

Neuromancer Neurostim is a program that can be used induce different mental states and moods, improve congnitive and mental abilities and analyze the effect of different pictures and stimulus to the user. Unlike typical brainwave entraintment software, Neuromancer Neurostim uses scientific measurements and advanced machine learning methods and models to generate stimulation sequences (pictures, keywords, audio) that will try to push brain towards target state. This is similar to the way art, music and exciting movies can change watcher's mood.

What it can do?

Neuromancer Neurostim shows and generates real-time videos based on measurements and computational models that keeps brain in a certain mood, changes brainwaves or exercises brain's cognitive abilities. After the stimulation sequence, the program reports brain measurements based numbers telling how well program stayed in target brain states.

In practice, the user selects:

  1. folder containing pictures and optional text file with keywords,
  2. program measures how user responds to pictures and
  3. after measurements and calculation of a prediction model, user can use program to generate videos according that try to reach target state as well as possible.

Installation & Requirements

Neuromancer Neuromancer requires 64bit Windows operating system (version 7 or later) and 64bit Java (version 7 or later) (www.java.com) to run. Earlier versions of the software might work too but you use them at your own risk.

You can use Neuromancer Neurostim without a brainwave measurement device like (Emotiv Insight) or Interaxon Muse by using precalculated models but for optimum performance one should buy and connect brainwave measurement device to the program.

Additionally, during a stimulation program Neuromancer keeps all pictures and other media files in memory meaning that you need to have enough RAM for this. For normal and heavy use, 8 GB of RAM and fast modern CPU is recommended.
